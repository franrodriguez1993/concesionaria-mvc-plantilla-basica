# LISTAR REPUESTOS

Verbo HTTP: GET
url : https://concesionariafranco.onrender.com/repuesto

Devuelve una lista de repuestos almacenadas en la base de datos.

Ejemplo:

```

[
  {
    "id_repuesto": 1,
    "nombre": "tuerca",
    "precio": 1000,
    "stock": 100,
    "id_proveedor": 3
  },
  {
    "id_repuesto": 2,
    "nombre": "cubierta",
    "precio": 100000,
    "stock": 100,
    "id_proveedor": 4
  }
]

```
